Our goal

The SWF (Static Website Foundation) goal is to bring back the original World Wide Web. Let's stop with all the nonsense of CSS, JavaScript and dynamic web pages. We want to bring back the feel of a real web page and the speed and easiness to reach the needed information without loosing yourself in an endless sea of buggy and slow user interfaces.

Why

  • It's more secure: with no code being run neither on the server nor on the client there's no threat for for any vulnerability (injections, XSSs, request forgeries, ...).
  • It's faster: the pages are smaller and less cluttered, so the download time will reduce. Furthermore, no calculations needed both serverside and clientside speed up the process even more.
  • It's more compatible: every device and web browser can render basic HTML. It can be seen by everyone on every device, no matter the hardware limitations.
  • It's easier: "How to center a div?", "How does async await work? with basic HTML you don't need to spend hour trying to figure out every trivial detail. So you can focus on what's really important: the backend.
  • It's the original: This is how Sir Tim Berners Lee first envisioned the web, and we all know that the original recipe is always the best recipe.
